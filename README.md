# UMS

## About
UMS is an Unreal Engine 5.4 project utilizing the Enhanced Input system and SteamCore Online Subsystem. It features stylized nature environments and supports multiplayer functionality via Steam.

## Getting Started

### Prerequisites
- **Unreal Engine 5.4**
- **Visual Studio 2022** (or compatible C++ compiler)
- **Steam Client** (running for multiplayer testing)

### Installation
1. Clone the repository.
2. Right-click `UMS.uproject` and select **Generate Visual Studio project files**.
3. Open `UMS.sln` and build the project.
4. Launch the project using `UMS.uproject`.

## Features
- **Engine**: Unreal Engine 5.4
- **Online Subsystem**: SteamCore (AppID: 480 - Spacewar) being used for development.
- **Input**: Uses Unreal's Enhanced Input System.
- **Assets**: Includes a "Stylized Nature Set" for environment creation.

## Plugins
The following plugins are enabled in this project:
- **ModelingToolsEditorMode**
- **BlockoutToolsPlugin**
- **BlueprintAssist**
